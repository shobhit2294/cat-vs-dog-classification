# 🐱🐶 Cat vs Dog Image Classifier

This project is a **Deep Learning based Cat vs Dog image classifier** built using **TensorFlow/Keras** and deployed with a **Flask API and simple HTML frontend**.

Users can upload an image and the model predicts whether it is a **Cat 🐱 or Dog 🐶**.

---

## 🚀 Features

- Image classification using CNN
- TensorFlow/Keras trained model
- Flask backend API for prediction
- Simple HTML frontend for image upload
- Real-time prediction

---

## 🧠 Model Details

- Framework: TensorFlow / Keras
- Architecture: Convolutional Neural Network (CNN)
- Input size: `128 x 128`
- Output: Binary classification (Cat or Dog)

## ▶️ How to Run the Project

### 1️⃣ Clone repository
git clone https://github.com/shobhit2294/cat-dog-classifier.git

cd cat-dog-classifier


### 2️⃣ Install dependencies
pip install tensorflow flask pillow numpy


### 3️⃣ Run backend server
python backend/app.py


Server will start at: http://127.0.0.1:5000


### 4️⃣ Open frontend

Open `frontend/index.html` in your browser.

Upload an image and click **Predict**.
